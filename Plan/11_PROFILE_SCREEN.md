# 11 — Profile Screen

[← Back to plan index](MOBILE_APP_PLAN.md)

## Goal

Central place for the user to see identity (local-only), progress, and app settings.

---

## Profile Content (MVP)

- Display name (editable locally)
- Total points
- Current streak
- Earned badges (grid/list)
- App info (version, course/project info)

---

## Settings (MVP)

- Toggle: Haptics on/off
- Toggle: Reduce motion
- Action: Reset demo data (with confirmation)
