# 09 — Challenges Screen

[← Back to plan index](MOBILE_APP_PLAN.md)

## Goal

Weekly challenges provide short-term goals that connect scans + learning to a feeling of progress.

---

## Challenge Types (Initial)

Auto-tracked:

- “Scan 5 items this week”
- “Scan 3 different waste types”
- “Complete 2 quizzes”

Self-reported (optional / stretch):

- “Avoid plastic bags for 7 days” (manual check-in)

---

## UX Requirements

- Show active challenges as cards with:
  - title + short description
  - progress bar
  - reward preview (points/badge)
- Provide “Completed” state with subtle celebration (animation/haptic)

---

## Acceptance Criteria (MVP)

- At least 3 predefined challenges ship offline.
- Progress updates from:
  - confirmed scan
  - quiz completion (if implemented)
